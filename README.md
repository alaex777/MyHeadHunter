# MyHeadHunter
## Установка
- Установить python3, docker
- Выполнить:
- docker compose up
- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- cd my_head_hunter
- python manage.py makemigrations
- python manage.py migrate
- python manage.py runserver
